# App_Jovem_Aprendiz
Projeto Web de site focado no cadastro de talêntos para o projeto Jovem Aprendiz. 

## Foco
O projeto tem o foco de registrar e acompanhar o desempenho do estudante durante sua permanencia no programa Jovem Aprendiz. Tendo o cadastro do aprendiz e daquele responsavel por monitora-lo.

# Ferramentas
No projeto foi usado: JavaScript, Node e MongoDB.
